## 2. 诊断数据明细 (The Evidence)
**文件**: `Results/validation_results/season_summary.csv`
- **结构**: 每行对应一个赛季 (`season`)。
- **关键字段**:
  - `max_rhat`: 该赛季所有参数中最大的 R-hat 值。
  - `min_ess`: 该赛季所有参数中最小的有效样本数。
  - `accuracy`, `top2_accuracy`: 该赛季的具体预测表现。
  - `coverage_90`: 该赛季的校准水平 (Calibration Level)。

## 4. 可视化与复现代码 (The Lens)
**脚本**: `Task1_VoteInference/task1_visualize.py`
- **功能**: 生成以下图表：
  - `fig_convergence`: R-hat 和 ESS 的分布直方图。
  - `fig_prediction`: 准确率随赛季变化的柱状图。
  - `fig_calibration`: 预测概率 vs 真实频率的校准曲线 (Calibration Curve)。
  - `fig_uncertainty`: 不确定性随周次推移的动态变化。
- **说明**: 运行该脚本可复现论文中的所有核心可视化。

## 5. 模型定义与数学结构 (The Theory)
**脚本**: `Task1_VoteInference/task1_model.py`
- **内容**:
  - Likelihood 函数定义 (Softmax / Plackett-Luce)
  - 先验分布设定 (Priors)
  - 观测数据约束处理 (Constraints)
**脚本**: `Task1_VoteInference/task1_metrics.py`
  - Brier Score, Entropy 等指标的计算逻辑
