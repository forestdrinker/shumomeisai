## 🟢 Task 1: 粉丝投票估算 (Fan Vote Estimation)

**赛题要求:** 建立数学模型估算粉丝投票，并验证其准确性与确定性。

### 1. 核心方法 (Methodology)
我们采用了 **动态贝叶斯 Plackett-Luce 排序模型**。
*   **潜变量**: 假设选手的受欢迎程度 $u(t)$ 服从随机游走 (Gaussian Random Walk)。
*   **似然函数**: 利用历史上观测到的“淘汰事件”来反向推断隐含的粉丝投票份额 $v$。

### 2. 客观运行结果 (Empirical Results)
| 指标 (Metric)             | 测量值 (Value) | 解释 (Interpretation)                                                                                           |
| :------------------------ | :------------- | :-------------------------------------------------------------------------------------------------------------- |
| **预测准确率 (Accuracy)** | **86.46%**     | 模型成功预测了历史淘汰结果的 86.5%。这表明估算出的粉丝票数具有极高的可信度，并非随机猜测。                      |
| **校准度 (Brier Score)**  | **0.0742**     | 远优于随机基准 (0.25)。说明模型不仅预测对了“谁会走”，而且其输出的概率值（如“A有90%概率淘汰”）在统计上是校准的。 |
| **确定性 (Certainty)**    | **~0.20**      | 90% 置信区间 (CI Width) 平均宽度约为 0.2。这表明我们能将粉丝支持率锁定在一个相对精确的范围内，而非模糊的估计。  |

## 🔵 Task 2: 投票规则比较 (Voting Rule Comparison)

**赛题要求:** 比较“排名法”与“百分比法”的差异，并检查是否存在针对评委或粉丝的偏向。

### 1. 核心发现：评委的主导性 (The "Judge Dominance")
基于对原始数据的**第一性原理核查**，我们得出了颠覆传统认知的结论：

| 关键相关性 (Correlation) | 数值 (Spearman Rho) | 结论 (Conclusion)                                                                |
| :----------------------- | :------------------ | :------------------------------------------------------------------------------- |
| **评委 vs 最终结果**     | **0.9797**          | **极强主导**。在 98% 的情况下，评委分数直接决定了比赛走向。评委掌控全局。        |
| **粉丝 vs 最终结果**     | **0.2323**          | **微弱影响**。目前规则下，粉丝投票与最终结果的相关性极低。粉丝几乎沦为“气氛组”。 |

### 2. 规则对比 (Rank vs Percent)
模拟显示，两种现有规则并没有本质区别：
*   **Rank Rule**: 粉丝相关性 0.92 (模拟值)，爆冷率 16%。
*   **Percent Rule**: 粉丝相关性 0.93 (模拟值)，爆冷率 22%。
*   **结论**: 无论哪种旧规则，都无法改变“评委吃肉，粉丝喝汤”的权力结构。百分比法虽然略微提升了粉丝权重，但也显著增加了比赛的随机波动（爆冷率上升）。

## 🟣 Task 4: 新系统设计 (New System Design)

**赛题要求:** 提出一个更"公平"（或更好）的新系统。

### 1. 设计哲学：赋权于民 (Empowering the Fans)
鉴于 Task 2 发现目前系统是“评委独裁” (Rho=0.98)，我们的目标不是“保护评委”，而是**“拯救粉丝话语权”**，同时维持起码的专业底线。

### 2. 推荐方案
我们通过多目标优化 (MOBO) 找到了 Pareto 前沿上的最佳折衷点：**混合规则 (Mixed Rule)**。

*   **参数设置**:
    *   **Gamma = 1.0**: 采用混合计分公式 $S_{total} = \text{Rank}(S) + \text{Rank}(v)$ (更稳定的排名逻辑)。
    *   **Eta = 1.92**: 引入“评委权重修正”，防止完全失控。

### 3. 预期效果 (Simulation Results)
| 指标                        | 旧系统 (Current) | **新系统 (New Proposal)** | 提升 (Improvement)                                               |
| :-------------------------- | :--------------- | :------------------------ | :--------------------------------------------------------------- |
| **评委相关性 (Fairness)**   | 0.98 (过高)      | **0.53**                  | **权力制衡**。评委不再能单手遮天，但仍保留 50% 的核心话语权。    |
| **粉丝相关性 (Engagement)** | 0.23 (过低)      | **0.53**                  | **参与感倍增**。粉丝的每一票都更真实地反映在结果中，相关性翻倍。 |
| **悬念度 (Drama)**          | 中等             | **High**                  | 由于权力的平衡，比赛结果将更难预测，增加了节目的观赏性。         |

## 📝 总结 (Overall Conclusion)

本项目的核心叙事链是逻辑闭环的：
1.  **模型验证**: 我们的粉丝投票估算模型准确率高达 86.5%，数据底座坚实 (Task 1)。
2.  **问题诊断**: 数据揭示了现有 DWTS 赛制实际上是**评委高度垄断**的 (Correlation 0.98)，粉丝影响力微乎其微 (Task 2)。
3.  **成因分析**: 粉丝投票不仅受选手表现影响，还带有强烈的**群体偏见** (如对选美选手的排斥)，这与评委的专业视角形成冲突 (Task 3)。
4.  **解决方案**: 我们提出的新系统 (Gamma=1.0) 并不是要推翻评委，而是要将粉丝的话语权从目前的“装饰品”地位提升到与评委**分庭抗礼 (0.5 vs 0.5)** 的水平，从而打造一个真正“全民参与”的现代综艺 (Task 4)。
