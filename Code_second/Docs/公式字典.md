## Indices
- $s\in\{1,\dots,34\}$：赛季索引（Season index）
- $t\in\{1,\dots,T_s\}$：周索引（Week index），$T_s$ 为赛季 $s$ 总周数
- $i$：选手（明星–舞伴组合）索引（Couple index）
- $j$：评委索引（Judge index），但题面说明 “Judge YYY” 并不对应同一人跨周/跨季
> 2026_MCM_Problem_C

## Sets
- $\mathcal A_{s,t}$：赛季 $s$ 第 $t$ 周仍在赛的选手集合（active set）
- $\mathcal E_{s,t}=\mathcal A_{s,t}\setminus \mathcal A_{s,t+1}$：第 $t$ 周淘汰集合（可空/可多人）
- $\mathcal F_s=\mathcal A_{s,T_s}$：决赛周 finalists 集合
- $\mathcal J_{i,t}$：该周该选手有效评委集合（去掉 N/A 评委）

## Observed variables (data)
- $s_{i,t,j}$：第 $t$ 周第 $j$ 位评委给 $i$ 的分数（可缺失 N/A）
- $S_{i,t}$：评委总分/均分（建议均分以消除评委人数变化）
- $p^J_{i,t}=\dfrac{S_{i,t}}{\sum_{k\in\mathcal A_{s,t}}S_{k,t}}$：评委百分比（week 内归一）
- $\text{placement}_i$：赛季最终名次（1 最好）
> 2026_MCM_Problem_C

## Latent variables (Task1)
- $u_{i,t}$：人气潜变量（popularity latent state）
- $v_{i,t}$：观众票份额（vote share），满足 $v_{i,t}\ge 0$、$\sum_{i\in\mathcal A_{s,t}}v_{i,t}=1$

## Rule regime indicators
- $m_s\in\{\text{rank},\text{percent}\}$：赛季规则类型（rank/percent）
- $\text{save}_s\in\{0,1\}$：是否启用 judges’ save（默认从 $s\approx 28$ 起，合理假设）
> 2026_MCM_Problem_C

## Differentiable rule mapping (Task1/2 internal)
- $\widetilde{\mathrm{rank}}_{\kappa}(\cdot)$：soft-rank 算子
- $\kappa_J,\kappa_F,\kappa_C>0$：soft-rank 温度参数（越小越接近硬 rank；需用 $\kappa\approx c/\Delta$ 指导并 CV 微调）
> Chatgpt 修改意见 最终版
- $C_{i,t}$：percent 合并下的周综合分（如 $p^J_{i,t}+v_{i,t}$ 或带权版本）
- $b_{i,t}$：淘汰“坏度”（越大越危险；用于软淘汰似然）

## Key hyperparameters (Task1)
- $\sigma_u^2$：人气随机游走周际波动强度（drift variance）
- $\lambda>0$：淘汰软似然尖锐度（越大越接近硬淘汰）
- $\gamma\ge 0$：judges’ save 中评委偏好强度（概率化二选一）
- $\tau_u^2$：初始人气方差（先验尺度）

## Validation & diagnostics
- $\widehat R$：Gelman–Rubin 收敛诊断（R-hat）
- $\text{ESS}$：有效样本量（effective sample size）
- $\text{ELBO}$：变分推断目标（仅用于 warm start/回退）
- $\text{PCR}=\dfrac{\mathrm{Var(prior)}}{\mathrm{Var(posterior)}}$：Posterior Contraction Ratio（可识别性量化）
> Chatgpt 修改意见 最终版
- $P(e_{s,t}=i\mid \text{data})$：淘汰事件后验预测分布（posterior predictive）
- $\mathcal C_{90}(s,t)$：90% 预测可信集合（最小累计概率集合）
- $\mathrm{Coverage}_{90}=\frac{1}{N}\sum_{(s,t)}\mathbf 1[\text{true elim}\in\mathcal C_{90}(s,t)]$：覆盖率检验
> Chatgpt 修改意见 最终版
- $\mathrm{Brier}$：淘汰事件概率预测的 Brier 分数
- **Acc / Top2**：淘汰复现率 / 是否落在预测 top2 危险人选中

## Task4 rule design & objectives
- $\theta=(a,b,\eta,\gamma,\text{save\_flag},\dots)$：候选赛制参数向量
- $w_t=\sigma(a(t-b))$：动态权重（logistic），$\sigma(\cdot)$ 为 sigmoid
- $\tilde v_{i,t}(\eta)=\dfrac{v_{i,t}^{\eta}}{\sum_k v_{k,t}^{\eta}}$：票份额变形（压缩/放大，控制公平/抗刷票）
- $P_\theta$：规则 $\theta$ 下的最终名次向量（placements）
- $\rho_S(\cdot,\cdot)$：Spearman 秩相关
- $Obj_F(\theta),Obj_J(\theta),Obj_D(\theta),Obj_R(\theta)$：四目标（观众对齐/技术对齐/戏剧性/鲁棒性）
> Chatgpt 修改意见 最终版
- $d_K(\cdot,\cdot)$：Kendall 距离（名次差异度量）
- $P_{\theta}^{pert}$：票扰动后的名次
- **建议记号隔离**：扰动强度写作 $\kappa^{pert}$（避免与 soft-rank 的 $\kappa$ 混淆）
